Welcome to My Docker Compose Project 🚀

Deploying a Static Website using Docker Compose & Nginx

This website is running inside an Nginx Docker Container managed by Docker Compose. The goal of this project is to learn how multiple Docker services can be managed using a single configuration file.

Learn Docker

📦 Docker

Containerizes applications so they run consistently across different environments.

⚙ Docker Compose

Starts, stops and manages multiple containers using one YAML file.

🌐 Nginx

A lightweight and high-performance web server used to host this website.
